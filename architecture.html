🚦 Smart Proxy

Receives every request. Validates the API Key against Redis (fast) or the Management Service (fallback). Only valid keys are forwarded.

Gateway :8080

⏱️ Rate Limiting

Uses a Redis Sliding Window algorithm to enforce per-minute limits defined by the API owner. Returns 429 when exceeded.

Management :8081

🔑 Key & Plan Management

API Owners register their backends, create custom pricing tiers (e.g. Gold: 500 RPM), and generate hashed API keys for their consumers.

Usage Service :8082

📊 Usage Analytics

Listens to RabbitMQ events, tracks monthly usage per key, and automatically disables keys that exceed their quota. Fires webhooks at 80% and 100%.
